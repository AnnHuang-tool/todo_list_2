## 安裝方式 
1. 開啟終端機，進入至欲儲存此專案的資料夾，並clone此專案

`git clone https://github.com/AnnHuang-tool/todo_list_2`

2. 進入專案資料夾

`cd todo-sequelize `

3. 在進入專案資料的狀態下安裝套件

`npm install`

4.安裝nodemon

`npm install nodemon`

5. 啟動程式

`npm run dev`

成功啟動後，終端機會顯示 Express is listening on localhost:3000

請在瀏覽器網址輸入` localhost:3000 `，即可開啟

*需要創建 .env.development 以便在本機端開啟伺服器
